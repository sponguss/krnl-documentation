# Fire Click Detector
```lua
<void> fireclickdetector(instance: ClickDetector, distance?=0, flag?="MouseClick")
```
Fires `instance` `ClickDetector`'s `flag` signal, with a provided distance

| Flags            |
|------------------|
| MouseClick       |
| RightMouseClick  |
| MouseHoverEnter  |
| MouseHoverLeave  |
